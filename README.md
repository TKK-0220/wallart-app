## サービス概要
部屋の壁や家具の幅を入力するだけで、黄金比に基づいた理想のサイズを診断するサービス。
算出した理想値に近い市販の額縁サイズを算出し、失敗を防ぐ。(本リリースまで)
壁の状況（家具の上、壁）に応じたロジックを実装し、それぞれ理想の値を割り出せるようにする。

## このサービスへの思い・作りたい理由
私自身にインテリアの知識やセンスがなく、壁ににお気に入りの写真を飾ろうとしたときに大きさやバランスに満足いかず、壁が画鋲の穴だらけになってしまった経験があります。
インテリアに関する情報は多くありますが、実際には「センス」や「感覚」に委ねられているものが多く、正解が分かりづらいと感じていました。

そこで、個人のセンスや感覚に頼らず誰でも同じ答えにたどり着ける指標として黄金比を使えば、
失敗を減らしてインテリアをもっと楽しめるのではないかと考えました。

## ユーザー層について
1.新生活を機に部屋にお気に入りの写真などを飾っておしゃれにしたい20〜40代の男女。
　年代：賃貸での新生活を始める20代をはじめ、ライフスタイルの変化に伴う住宅事情(子供の写真や思い出の写真を綺麗に飾りたいなど)に合わせた年代設定にしました。

2.インテリア初心派やセンスに自身がない人
　センスや感覚ではなく、理由や数字的なロジックがあると安心できる人。失敗したくない人。

## サービスの利用イメージ
1.トップページにて家具の上に飾るか壁に飾るかの選択を行う。
2.壁や家具の幅を測り、数値をフォームに入力する。
3.診断結果として以下を確認できる。
　1.視覚的に美しく見える理想のアートサイズ
　2.それに近い市販の額縁サイズ

## ユーザーの獲得について
「額縁　サイズ　選び方」や「ポスター 飾り方」などをの検索流入を想定。
開発者自らサービスを体験し、SNSなどでの投稿を行う。

## サービスの差別化ポイント・推しポイント
既存のサイトではシュミレーターが多く、感覚やセンスに委ねられるところが多く、あくまで試す場所となる。
また、黄金比計算ツールはすでに存在するが、そこから額縁サイズを探して導き出すのは面倒。
本アプリは、センスに自信がない人が答えを導き出す場所として、季節ごとの模様替えや写真が増えたときなどでも
すぐに確認できるツールとして活用してもらえたらと思う。

## 機能候補
MVP
・診断フォーム: 壁・家具の幅入力、飾る場所の選択。
・計算・判定ロジック: 黄金比を用いた推奨サイズの算出。

本リリース
・理想値に近い市販サイズをDBから抽出・表示。
・複数枚レイアウト: 黄金比の枠内に複数の写真を並べるパターンの提案。
